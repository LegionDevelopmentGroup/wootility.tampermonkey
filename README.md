# Wootility Tampermonkey Scripts

Make sure you have all of your Wootility profiles backed-up before using anything in this repo.

Only tested on a profile for a Wooting Two HE with basic single-layer Static "RGB Settings".

1) Add the Tampermonkey script.
1) Go to https://wootility.io/.
1) Make sure you have at least one Inactive profile. The Tampermonkey script will destructively modify the first Inactive profile.
1) Refresh the page.
1) Click the "🔵🟢 Gradient" button in the bottom-right corner of the page.
1) Refresh the page without clicking anything else.
1) Now, if you go to the first Inactive profile, the RGB Settings should be updated to have each row of keys a different color, in the pattern of a gradient from blue to green from top-to-bottom.
